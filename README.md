# mock-chatui

デモ用のチャットUIモックです。

## 機能

- 📱 スマートフォン・PC対応のレスポンシブデザイン
- 💬 画面下部に固定されたチャット入力欄
- 🟢 送信メッセージは右側に緑色の吹き出しで表示
- ⚪ 自動返信は左側に灰色の吹き出しで表示
- ⌨️ タイピングインジケーター付き
- 📜 スクロールで履歴を確認可能

## 使い方

`index.html` をブラウザで開くだけで動作します。

```bash
# ローカルサーバーで起動する場合
python -m http.server 8000
# または
npx serve .
```

## 自動返信メッセージ

20パターンの明るく陽気な日本語メッセージがランダムで返信されます。
